Правила спільноти CS2 MicroTwitter

Дотримання цих правил допомагає створити комфортну атмосферу для всіх учасників спільноти.
1. Загальні положення
  • 1.1. Використовуючи наш сайт, ви автоматично погоджуєтесь з цими правилами.
  • 1.2. Адміністрація має право змінювати правила без попередження.
  • 1.3. Мінімальний вік для реєстрації - 13 років.
  • 1.4. Один користувач - один акаунт.
2. Поведінка в спільноті
  • 2.1. Поважайте інших учасників спільноти.
  • 2.2. Заборонені образи, дискримінація та цькування.
  • 2.3. Не публікуйте особисту інформацію інших людей.
  • 2.4. Заборонений спам та флуд.
  • 2.5. Не створюйте кілька акаунтів для обходу блокувань.
3. Контент
  • 3.1. Весь контент має стосуватися CS2 або ігор загалом.
  • 3.2. Заборонено публікувати NSFW контент.
  • 3.3. Не порушуйте авторські права.
  • 3.4. Заборонено рекламу без дозволу адміністрації.
  • 3.5. Чіти та шахрайство заборонені.
4. Санкції
  • 4.1. Попередження за незначні порушення.
  • 4.2. Тимчасове блокування за повторні порушення.
  • 4.3. Постійне блокування за серйозні порушення.
  • 4.4. Видалення контенту, що порушує правила.
Контакти для скарг
