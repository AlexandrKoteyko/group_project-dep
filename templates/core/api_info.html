API інформація

Наш API знаходиться в розробці. Деякі ендпоінти вже доступні для тестування.
Доступні ендпоінти
Ендпоінт Методи Опис
/api/posts/ GET Отримання списку постів
/api/posts/<id>/ GET Отримання детальної інформації про пост
/api/users/ GET Отримання списку користувачів
/api/users/<id>/ GET Отримання інформації про користувача
/api/forum/topics/ GET Отримання списку тем форуму
/api/gallery/ GET Отримання списку медіа з галереї
/api/portfolio/ GET Отримання списку елементів портфоліо
Приклад використання
Отримання списку постів: 
fetch('/api/posts/')
    .then(response => response.json())
    .then(data => console.log(data));
Отримання конкретного поста:
fetch('/api/posts/1/')
    .then(response => response.json())
    .then(data => console.log(data));
Авторизація

Для доступу до захищених ендпоінтів потрібна авторизація через токен:

fetch('/api/protected-endpoint/', {
    headers: {
        'Authorization': 'Token ваш_токен'
    }
})
Обмеження
  • Ліміт запитів: 100 запитів на годину з однієї IP-адреси
  • Максимальний розмір відповіді: 1MB
  • Підтримуються формати: JSON
